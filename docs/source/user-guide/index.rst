User Guide
==========

This user guide provides practical, operator-focused documentation for managing and operating the Edge Gateway in production environments. It covers the core remote management features exposed through ThingsBoard and explains how to safely control, update, and maintain deployed devices without physical access.

The guide is intended for system operators, developers, and administrators who work with Edge Gateway deployments and require reliable tools for configuration management, software updates, and operational control.

Use the sections below to explore individual features in detail, including remote file synchronization, software updates, and on-demand remote actions via RPC.
